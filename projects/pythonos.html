Python OS

A unique environment for running Python and Pygame apps.

Python OS has been one of my longest running projects. It has slowly grown from being just a small window controlled by a few keys to a fully fledged environment with menus, software managers, and app repositories. Please note, however, that development of Python OS 5x has been delayed indefinitely because of a multitude of errors and lack of time.
